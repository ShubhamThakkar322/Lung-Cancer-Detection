🫁 Lung Cancer Detection using Histopathological Images
📌 Overview

This project focuses on detecting lung cancer from histopathological images using Deep Learning techniques. The model is trained to classify tissue images into different categories, helping in early and accurate diagnosis.

The dataset used contains high-resolution microscopic images of lung and colon cancer tissues.

📄  Shubham Thakkar

🚀 Features

🧠 Deep Learning-based image classification

🔬 Works on histopathological (microscopic) images

📊 High accuracy model for cancer detection

⚡ Efficient preprocessing and training pipeline

📁 Well-structured dataset handling

📈 Visualization of training performance

📂 Dataset

We used a publicly available dataset:

🔗 https://www.kaggle.com/datasets/andrewmvd/lung-and-colon-cancer-histopathological-images

📄 Research Paper: https://arxiv.org/abs/1912.12142v1

Dataset Details:

Images of lung and colon tissues

Classes include:

Lung Adenocarcinoma

Lung Squamous Cell Carcinoma

Benign Tissue

High-resolution RGB images
